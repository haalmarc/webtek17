2. Purpose, goals and audience

This website's purpose is, first of all to upgrade Trondheim Lindy Hops existing web page, and in doing so making TLH more accessible to the public. Our goal is the same as our clients; to invite people into the great world of the Lindy Hop dance. The audience we are trying to reach are those who have danced Lindy Hop previously and want to start dancing more frequently, and those who are new and curious as to what Lindy Hop is.

3. Navigation structure

The website will consist of 4 webpages, 3 of which will be accessible from all the other pages. This way all the information available is accessible from the entire website. The last webpage will be a signup form, which will only be reachable from the courses page, but will have the same navbar as the other webpages. Here is an organizational map of the website.

4. Page Layout and Appearance

In this section we will present the design decisions that are common across all webpages of the website. In the content section we have more specific information about the individual webpages and associated mockup images. We’ve had a focus on readability when choosing our design. Throughout the pages we want to create breathing room for the user, using a smaller width and using margins and padding. We have followed a standard text size to ensure readability. We’ve chosen contrasting colors to make the text more readable, as well as to accommodate the feel we want the page to give. Since Lindy Hop is a dance style from the early 19th century we have chosen black and white pictures to give the site a vintage look.

Globals

Width 70%. Use 16px as standard font size and font type Georgia. Use background color white. Use text color black.

Header

Width: 100 %. Background color #985452. White text color. Logo on the left, 200px image. Navigation bar elements with equal distance, center aligned with the logo. Text size 50px. Margin: 20 px. Padding: 20px.

Footer

Width: 100 %. Background color #985452. White text color. Text size 10px. Text vertically and horizontally aligned. Instagram and Facebook icons in size width 20px center aligned above the text.

5. Content

Homepage

This page will provide an overview about what resources you can find on the page. From here you can orient yourself on how to find information about the group and the dance lindy hop, scheduled courses, events and also find a translated version of the page. The page should give a clear view of what information you can find, through having a simple and easily readable page. By using Javascript the background image will slowly change in a slideshow. By doing this we hope that the page will feel more inspiring, and will make the user want to stay on the page longer to view the different pictures of the dance.

About

This section is going to give the user some information about our client. This includes a short introduction to what kind of group our client is, group history, some information about the services our client provides, goals, general information about events and courses, facebook page, physical address as well as the names of some of the central figures in our clients hierarchy.

The page consists of two parts, excluding footer and header: the title “Om oss” and the about us text. The title will be above a background image of people dancing. Image has 100% width. The about us text will have a white background and be center aligned.

Courses

This section will include the scheduled beginner and advanced courses for TLH. The page will help members find information on where and when to meet up. The section will also include information about what to expect from the course, so the potential member can choose what suits them best. On the bottom of the page there will be a link to a registration page where the user can sign up for a course.

We are going to use two headings, sizes 50px, presenting the two possible courses. The first heading and description are going to be on the left side of the page, followed by a round picture on the right side (300px width). The second heading and description, we will put accordingly on the right side, placing a picture on the left side, here again 300 px width. On the bottom of the page we will include a sign up button referring the user to “påmeldingsskjema”.

Sign up

The sign up page is where the user can register him/herself for a course or an event. It will contain a form with boxes for:

  • First name
  • Last name
  • Email
  • If you are a beginner or have danced before
  • Phone number
  • Where you heard of TLH
and checkboxes for membership and ToS, as well as a submit button.

Events

By clicking on “Events” the client is going to be referred to the facebook event page of Trondheim Lindy Hop. The events page will contain information about courses and events. The facebook calendar is frequently updated, providing the user with up to date information.

6. Minimum Requirements

The background of the homepage will consist of a slideshow programmed in JavaScript. Since we want to inspire people to dance we want to choose appealing pictures of people dancing. The slideshow will make the page more varied, but in order to not overwhelm the user the slideshow will have to transition slowly. The JavaScript will be in the file slideshow.js.

The footer and header on all the pages are going to look the same. We will use JS to implement these sections into each html file. This will be written in the header.js and footer.js files respectively.

On the sign up page there will be a submit button. When clicked, a box will pop up and tell the user if the submission was successful or not. This will be written in script.js

Because some of our audience speak english, we are going to give the user an option to translate the page. This will be done through Google Translate and JS. This will be written in the script.js file.

If we have time we also hope to add a facebook calendar on a separate "Events" page. This will change a lot of the structure of the website, but make the website more userfriendly by not having to navigate the user off the website.
